## Docker 練習專案
### 簡介
### 此專案是我學習 Docker 的練習記錄，包含基礎環境設置、容器管理、應用部署以及多容器協作等範例，旨在提升對容器化技術的理解與實踐能力。

### 練習內容

1. 基礎操作
* 建立與運行容器 (docker run)
* 使用 Dockerfile 製作自訂映像檔 (docker build)
2. 容器化應用
* 部署簡單的 python flask 應用
* 部屬flask、redis架設網站和記錄資訊
3. 持久化與網路設定
* 建立具備資料持久化的容器 (Volumes)
* 配置 Docker 網路 (Bridge)
4. 多容器協作
* 使用 docker-compose 部署多容器應用
* 設置負載平衡與服務間通訊
